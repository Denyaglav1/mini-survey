# Промпты, использованные при создании проекта

## Промпт 1 — Постановка задачи и создание проекта

> Нужно создать мини-анкету: backend на Python (FastAPI) + frontend на чистом HTML/JS. Backend отдаёт GET /questions (3–5 вопросов) и принимает POST /answers. Frontend загружает вопросы, показывает форму, отправляет ответы и выводит «Спасибо!».

**Что сделал ассистент:**

1. Создал структуру проекта:
   ```
   mini-survey/
   ├── backend/
   │   ├── main.py
   │   └── requirements.txt
   ├── frontend/
   │   └── index.html
   ├── .gitignore
   └── README.md
   ```
2. Инициализировал git-репозиторий (`git init`)
3. Сгенерировал `backend/main.py` — FastAPI-приложение с:
   - CORS-middleware (allow_origins=["*"]) для доступа с фронтенда
   - 5 жёстко заданных вопросов (текст, число, выбор из списка)
   - Pydantic-модели для валидации входящих данных (`AnswerItem`, `AnswersPayload`)
   - In-memory хранилище (`list[dict]`) для ответов
   - 3 эндпоинта: `GET /questions`, `POST /answers`, `GET /answers`
4. Сгенерировал `backend/requirements.txt` с зависимостями:
   ```
   fastapi==0.115.0
   uvicorn==0.30.0
   ```
5. Сгенерировал `frontend/index.html` — одностраничное приложение:
   - Загружает вопросы через `fetch()` с backend API
   - Динамически рендерит форму (text input, number input, select)
   - Отправляет JSON с ответами через `POST /answers`
   - Показывает экран «Спасибо!» при успехе, ошибку при сбое
6. Создал `.gitignore` (исключены `__pycache__`, `.env`, `venv/`)
7. Написал `README.md` с инструкцией по запуску

**Результат:** Полностью рабочий full-stack проект, готовый к локальному запуску.

## Промпт 2 — Установка зависимостей и запуск

> Запусти локально всё это

**Что сделал ассистент:**

1. Установил Python-зависимости:
   ```bash
   cd backend && pip install -r requirements.txt
   ```
2. Запустил backend-сервер:
   ```bash
   uvicorn main:app --host 127.0.0.1 --port 8000
   ```
3. Запустил frontend-сервер:
   ```bash
   cd frontend && python -m http.server 3000
   ```
4. Обнаружил проблему с прокси на машине (все запросы к localhost возвращали 503)
5. Диагностировал: заголовок `Proxy-Connection: close` указал на системный прокси
6. Решение: обход через переменную окружения `NO_PROXY=127.0.0.1,localhost`
7. Проверил что оба сервера отвечают корректно

**Результат:** Backend работает на `http://localhost:8000`, frontend на `http://localhost:3000`.

## Промпт 4 — Шутливые валидации полей

> Добавь шутливые валидации: если человек ввёл имя — подписывать «Что-то я не уверен». Сколько лет — подбери диапазоны зумер, миллениал и т.д., выдавай под полем «понятно — зумер» или кто подходит. Если опыт программирования больше 2 лет — пиши «Нам-то не гони».

**Что сделал ассистент:**

1. Добавил 2 новых вопроса в backend: «Сколько вам лет?» и «Сколько лет вы программируете?»
2. Реализовал 4 валидатора на frontend (real-time, при вводе):
   - **Имя** — скептическая реакция на любой ввод: «Что-то я не уверен...», спецкейсы для 1 буквы, чисел, admin/root, длинных имён
   - **Возраст** — определение поколения:
     - ≤12 → Альфа («в Roblox уже деплоишь»)
     - 13–19 → зумер
     - 20–27 → зумер со стажем
     - 28–35 → миллениал («помнишь jQuery?»)
     - 36–45 → миллениал-олд («верстал таблицами»)
     - 46–55 → поколение X
     - 56–70 → бумер
     - 71+ → мудрец / бессмертный
   - **Опыт программирования** — 0–2 года верим, >2 → «Нам-то не гони!» с нарастанием
   - **Уровень вайбкодинга** — от «открыл терминал — закрыл» до «ДЕПЛОЮ ВЗГЛЯДОМ»
3. Подсказки появляются под полем в цветных плашках (жёлтая / розовая / зелёная / красная)

**Результат:** Форма реагирует на ввод в реальном времени с шутливыми комментариями.

## Промпт 5 — Редизайн в стиле брутализма

> Можешь задизайнить форму в стиле брутализма

**Что сделал ассистент:**

1. Полностью переписал CSS в эстетике веб-брутализма:
   - Моноширинный шрифт **Space Mono** (Google Fonts)
   - Жирные чёрные рамки 3–4px, `border-radius: 0` везде
   - Жёсткие тени со смещением (`box-shadow: 12px 12px 0 #000`)
   - Кислотно-зелёный акцент `#CDFF50` на чёрном фоне
   - Бежевый фон страницы `#f5f0e1`
   - Вся типографика — UPPERCASE с letter-spacing
2. Интерактивные элементы:
   - Кнопка с эффектом «вдавливания» (shadow уменьшается при hover/active)
   - Инпуты подсвечиваются зелёным при фокусе
   - Выбранная радиокнопка — инверсия: чёрный фон + зелёный текст
3. Детали:
   - Метка `[ FORM_v2.0 ]` над карточкой
   - Плейсхолдеры в стиле `/// ваш ответ`
   - Нумерация вопросов — чёрные квадраты с зелёными цифрами
   - Хинты валидации — цветные плашки без скруглений с `>` префиксом

**Результат:** Форма выглядит как брутальный арт-объект — грубая типографика, контрастные цвета, нулевые скругления.
