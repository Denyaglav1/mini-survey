>
> **Домашнее задание: Генерация простого приложения в IDE**
>
> Цель: создать минимальное full-stack приложение (backend + frontend) с помощью AI-ассистента в IDE; настроить API для получения конфигурации анкеты и отправки ответов; научиться просить ассистента структурировать архитектуру и генерировать отдельные файлы; развернуть фронтенд, подключенный к backend API; реализовать хранение пользовательских ответов в памяти backend-приложения; проверить работу приложения локально.
>
> Backend предоставляет два API:
> - GET /questions — возвращает список вопросов анкеты (жёстко заданных, 3–5 шт.).
> - POST /answers — принимает ответы пользователя и сохраняет их в памяти.
>
> Frontend: загружает вопросы с backend; отображает их в HTML/JS интерфейсе; отправляет заполненные ответы через POST /answers; показывает пользователю сообщение «Спасибо!» после отправки.
